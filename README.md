# App ToDo - Django

## Descripción
Aplicación de gestión de tareas (ToDo) desarrollada con Django. Permite a los usuarios crear, editar y eliminar tareas, así como marcar tareas como completadas. Vamos a implementar 3 vistas: una para mostrar todas las tareas, otra para crear una nueva tarea y una última para editar una tarea existente.

## Pasos
1. Crear un nuevo proyecto de Django.
2. Crear una nueva aplicación dentro del proyecto.
